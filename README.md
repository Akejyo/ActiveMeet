How to set up:
Download the project and run:
    npm install
To install all the corresponding packages/modules.

Then, run:
    npm run seed
To run the seed file and seed the database with our testing data.

Then run:
    npm start
To begin running the server.
You can now access it through http://localhost:3000

For testing:
You will not be able to access any of the application's features until you sign in or create an account.
You can either create your own account or use one of our testing accounts.
